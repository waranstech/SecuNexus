Deployment

Deployment Options

Choose the deployment model that matches your organization's security and compliance requirements.

Full Air-Gap

Complete isolation from any network. Zero external connectivity required at any point.

  • No internet required — ever
  • USB-based encrypted updates
  • Offline vulnerability database
  • Local-only report generation
  • Signature-verified packages
Ideal for: Defense, Government, Critical Infrastructure

On-Premises with Controlled Updates

Run on your infrastructure with optional controlled connectivity for updates only.

  • Your data center, your control
  • Optional scheduled update windows
  • Proxy-compatible updates
  • Full data sovereignty
  • Flexible network policies
Ideal for: Banks, Insurance, Healthcare

Private Cloud

Deploy in your private cloud infrastructure (AWS VPC, Azure VNet, GCP VPC).

  • Private cloud compatible
  • VPC / VNet deployment
  • Container-based architecture
  • Auto-scaling support
  • Cloud-native integrations
Ideal for: Tech Companies, SaaS Providers, Startups

Enterprise

Enterprise-Grade Features

Built for organizations that need security, scalability, and compliance at scale.

Multi-Project Management

Manage hundreds of projects with team-based organization, shared policies, and centralized reporting across your entire portfolio.

Policy Engine

Define custom security policies and quality gates. Automatically block builds that violate severity thresholds or compliance requirements.

Audit Trail

Complete, immutable audit logs for every action — scans, findings, status changes, user activities, and configuration modifications.

Custom Rules

Create organization-specific security rules for SAST and SCA. Enforce internal coding standards alongside industry best practices.
